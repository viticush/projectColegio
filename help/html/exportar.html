Ventana exportar backup

En esta ventana podemos exportar todos los datos de BBDD a un fichero.

Lo primero que tenemos que hacer es seleccionar que tabla de base de datos queremos guardar sus datos

Cuando tenemos seleccionada la tabla deseada, le damos aceptar y tenemos que introducir un nombre y donde queremos almacenar el fichero.

El fichero guardado tendra como nombre, "nombredado_nombreTableSeleccionada_fecha.sql"
